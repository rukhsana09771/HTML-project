What is HTML?

HTML stands for HyperText Markup Language. It's markup language developed by sir Tim-Berners Lec in 1993 to develop web pages

HTML consists of elements that describes different type of content: Headings, images, links, paragraphs, buttons, forms, tables, videos etc.
